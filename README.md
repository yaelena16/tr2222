## Описание программы

`Expense Tracker` — это десктопное GUI-приложение для ведения учета личных финансов. Оно позволяет добавлять записи о расходах, указывая сумму, категорию и дату. Приложение также предоставляет инструменты для анализа данных, включая фильтрацию по периоду и категории, и автоматически подсчитывает общую сумму отфильтрованных расходов.

### Основные возможности:

*   **Добавление расходов:** Простая форма для ввода суммы, выбора категории и даты.
*   **Просмотр данных:** Все расходы отображаются в удобной таблице.
*   **Фильтрация:** Возможность отфильтровать записи по диапазону дат и по конкретной категории.
*   **Подсчет суммы:** Автоматический расчет и отображение общей суммы расходов для отфильтрованных данных.
*   **Сохранение и загрузка:** Все данные автоматически сохраняются в файл `expenses.json` и загружаются при запуске приложения.
*   **Проверка ввода:** Приложение проверяет корректность вводимых данных (сумма должна быть числом, дата — в верном формате).

## Технологии

*   **Язык:** Python 3
*   **GUI:** `tkinter` (встроенная библиотека), `tkcalendar` (для виджета выбора даты).
*   **Формат данных:** JSON для хранения расходов.
*   **Система контроля версий:** Git

## Как использовать

1.  **Клонируйте репозиторий** на свой компьютер.

2.  **Установите зависимости:**
    Приложению требуется библиотека `tkcalendar`. Установите ее через pip:
    ```bash
    pip install tkcalendar
    ```

3.  **Запустите приложение:**
    Выполните в терминале команду:
    ```bash
    python expense_tracker.py
    ```

## Примеры использования

*   **Добавление расхода:**
    1.  Введите сумму в поле "Сумма" (например, `150.50`).
    2.  Выберите категорию из выпадающего списка (например, "Еда").
    3.  Выберите дату в календаре.
    4.  Нажмите кнопку "Добавить расход". Запись появится в таблице.

*   **Фильтрация расходов:**
    1.  Чтобы увидеть расходы за определенный период, выберите начальную и конечную даты в полях "От" и "До".
    2.  Чтобы отфильтровать по категории, выберите ее из выпадающего списка в секции "Фильтры".
    3.  Нажмите кнопку "Применить фильтр". Таблица обновится, и в поле "Итого" будет показана сумма только для отображенных записей.
    4.  Чтобы сбросить фильтр, вы можете выбрать категорию "Все" и очистить поля с датами, после чего снова нажать "Применить фильтр".
